
# Candidate Rules for the Parity Problem in Cellular Automata

This repository contains the candidate rules identified in the paper:

**Exploring the Rule Space for the Parity Problem in Cellular Automata**

## Description

We consider one-dimensional binary cellular automata with radius 3.  
After a structured reduction of the rule space and computational filtering, 20 candidate rules were obtained.

## Data

- `rules.csv` — list of candidate rules (Wolfram codes)
- `diagrams/` — space–time diagrams for all rules

## Diagrams

The diagrams are provided for two initial configurations:

- **config_A**:  
  `11010100011101010010010000101`

- **config_B**:  
  `11010100011101010110010000101`

These configurations differ by a single bit.

For each configuration, diagrams for all 20 candidate rules are provided.

## Purpose

The diagrams illustrate the dynamical behavior of candidate rules and were used in the filtering process described in the paper.
